TrashParty

Visit TrashParty
GitHub Repository

NFT-inspired Webapp for (entertainment purposes): A website where once a day you open a pack of random stuff (data we collect from different API's). In the packs could be anywhere from songs to movies to pokemon cards, really any type of data we can get from an API and have time to implement (currently only GIFs from GIPHY). You can trade your "trash" with other users till you get items you actually like. TrashParty is a full-stack application that uses front and back end JavaScript to send and retrieve data. Some other technologies used are node.js, sequelize and MySQL for the database.
